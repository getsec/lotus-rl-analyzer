## install



```
# Clone the repo
git clone https://github.com/getsec/lotus-rl-analyzer && cd lotus-rl-analyzer

# Create virtualenv 
python3 -m venv .venv
source .venv/bin/activate

# install depends
pip install -r requirements.txt

# Copy over .env.example to .env
# Add your ballchasing API key to the file.

# run app
python main.py
```
